立即执行函数

【问题】在全局上声明的函数，保存在GO里，函数执行完成之后，还连着GO，没有释放，所以随时都可以调用函数，不需要重新定义。

【需求】如果一个函数就执行一次，之后就不需要了，基于这个问题，出现了这样的需求，希望函数执行完成之后立即释放，然后就有了立即执行函数

立即执行函数，（功能名称：初始化函数）(IIFE, immediately-invoked function expression)
【功能】自动执行，执行完成之后立即释放

自动执行，执行完成之后立即释放

参数

将立即执行函数赋值给变量，就可以接受立即执行函数的返回值

探究立即执行函数写法的本质原因

【立即执行函数写法的本质原因：一定是表达式才能被执行符号执行】
函数声明变成表达式的方法有()、+、-、！、(让左边为真)&&、(让左边为假)||，函数声明的函数名会被自动忽略
【立即执行函数会自动忽略函数名】
